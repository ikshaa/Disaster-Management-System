The Problem

During major disasters, emergency services are flooded with thousands of calls and reports simultaneously. Human operators cannot quickly triage which situations are life-threatening. Every second of delay costs lives.

How It Works
1
Citizen submits a report — text description, GPS location, and an optional photo of the disaster site
2
AI analyzes the text (DistilBERT NLP) and the image (ResNet50 vision) simultaneously
3
Priority score computed from NLP confidence, image severity, and GPS cluster density
4
Dashboard updates live — responders see AI-ranked alerts on an interactive map and dispatch help instantly

Key Features
🧠
Dual AI Engine
DistilBERT NLP classifies crisis text into 7 severity levels. ResNet50 vision detects fire, floods, collapses from photos.
⚡
Real-time WebSocket Dashboard
Reports appear live on an interactive Leaflet map as they arrive. No page refresh needed.
📍
Location-aware Clustering
Multiple reports from the same area amplify the priority score automatically. Haversine 500m radius detection.
📡
Offline Mesh Network
Works without internet. Relay nodes collect reports via local WiFi, cache in SQLite, sync to hub when connected.
🔴
AI Reasoning Transparency
Every report shows full AI breakdown: NLP class + confidence, image damage class, score formula, cluster info.
